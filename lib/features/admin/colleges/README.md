# College Management (Admin)

This module contains all screens, models, and services for managing colleges as a Super Admin.

## Features
- Add new colleges (name, location, code, etc.)
- List all colleges
- Edit and delete colleges

## Structure
- `screens/` — UI screens for listing, adding, and editing colleges
- `models/` — College data model
- `services/` — College data service (CRUD)
- `widgets/` — Reusable widgets for college management
